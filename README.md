# 킵탭 · Keep Tap ⌨️✨

> 투명 스위치 키캡을 톡톡 두드리는 힐링 토이.

요즘 유행하는 **투명 스위치 + RGB 아크릴 베이스 + 키캡** 키링 토이를 웹앱으로 옮긴 힐링 사이트입니다.
키캡을 누르면 **① 기계식 스위치 사운드 · ② 손끝 진동(모바일) · ③ 화면 흔들림 · ④ RGB 플래시**가 동시에 일어나요.

## 실행 방법

### 방법 1 — 그냥 열기 (가장 쉬움)
`index.html` 파일을 더블클릭하면 브라우저에서 바로 열립니다.
> 3D 라이브러리(Three.js)와 한글 폰트를 CDN에서 불러오므로 **인터넷 연결**이 필요합니다.
> (연결이 안 되면 "3D 엔진을 불러오지 못했어요" 안내가 표시됩니다.)

### 방법 2 — 로컬 서버 (권장)
```bash
npx serve -l 5173 .      # Node
# 또는
python -m http.server 5173
```
그 후 `http://localhost:5173` 접속.

## 무엇을 할 수 있나

- **누르기** — 화면의 스위치 키캡을 톡톡. 누를 때마다 RGB가 번지고, 소리·진동·흔들림이 와요.
- **키보드** — 스페이스·엔터·아무 글자, 또는 숫자 1~4로 해당 스위치를 누를 수 있어요.
- **Design (우측 상단)** — 나만의 토이로 꾸미기.

## 🎨 Design — 커스터마이징

우측 상단 **✦ Design** 버튼을 누르면:

| 항목 | 설명 |
|------|------|
| **슬롯 구성 (최대 4)** | 스위치를 1~4개로. 슬롯의 ⊕/✓ 버튼으로 켜고 끄고, 슬롯을 눌러 그 슬롯을 꾸며요 |
| **키캡 모양** | 기본 · 베개 · 🍞식빵(토스트 그라데이션) · 🐾고양이발(핑크 발바닥) · 높은 · 낮은 — 스펙 시트 기반 6종 |
| **키캡 색 / 투명** | 파스텔 색 또는 직접 선택(🎨), "투명 키캡"을 켜면 스위치가 비치는 클리어 |
| **스위치 색 (스템)** | 십자 스템 색 |
| **RGB 언더글로우** | 무지개 순환 ON, 또는 끄고 단색 선택 |
| **각인 (글자·이미지)** | 키캡 윗면에 글자(A, ♥, 쉼 …) 입력 또는 **내 이미지 업로드** |
| **3D 파츠** | 키캡 위에 ⭐별 · 💗하트 · ⭕링 · 🌸꽃 · 👑왕관 부착 |

> 모든 구성은 자동 저장(localStorage)되어 다음에 열 때 그대로 복원돼요.

## 기능

| 감각 | 설명 |
|------|------|
| 🔊 **촉감 사운드** | 실제 키캡 녹음(`keycap_sound_001.mp3`)을 base64로 내장해 즉시 재생. 매 탭 미세한 피치 변화로 자연스러움. (오프라인/더블클릭에서도 동작) |
| 📳 **진동** | 모바일 `navigator.vibrate`로 키캡마다 다른 햅틱 (미지원 기기는 자동 비활성) |
| 💥 **화면 흔들림** | 트라우마 기반의 잔잔한 카메라 흔들림 (세기 조절) |
| 🌈 **RGB 언더글로우** | 스모키 아크릴 베이스 안에서 빛이 번지고, 누를 때 플래시 |
| ✨ **연출** | 누를 때 베이스 둘레로 퍼지는 빛 물결, 떠다니는 입자, 격려 문구, 호흡 가이드 |

## 접근성
- `prefers-reduced-motion` 존중 — 자동 부유·패럴랙스·흔들림·입자 움직임을 끔
- 모든 컨트롤 키보드 조작 가능(`role="button"` + Enter/Space), 캔버스도 포커스 가능
- 진동 미지원 기기는 햅틱 자동 비활성(사용자가 끄지 않는 한 설정 보존)
- 저사양/모바일에서는 무거운 굴절(transmission) 대신 가벼운 반투명으로 자동 대체

## 기술
- **Three.js** (r0.169) — 실시간 3D, 환경맵(PMREM), 소프트 섀도, 스모키 반투명 재질
- **Web Audio API** — 스위치 사운드 합성 + 리버브 + 앰비언트 패드
- **Vibration API** — 모바일 햅틱
- 단일 `index.html`, 빌드 과정 없음
